## Quick Reference

### Applications

| Application | Command | Purpose |
|------------|---------|---------|
| Interactive Debugger | `python interactive_debugger.py` | Step-by-step execution |
| Basic Visualizer | `python dfa_visualizer.py` | Test and visualize |
| Core Library | `python dfa.py` | Test core functionality |

### Example DFAs

| File | Language | Description |
|------|----------|-------------|
| `even_a_dfa.json` | Even 'a's | Strings with even number of 'a's |
| `ends_with_ab.json` | Ends "ab" | Strings ending with "ab" |
| `divisible_by_3.json` | Div by 3 | Binary numbers divisible by 3 |
| `odd_b_dfa.json` | Odd 'b's | Strings with odd number of 'b's |
